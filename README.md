
# ToDo-List

This is ToDo-List made using mostly javascript and also html and css. it is based on CRUD operation i.e. create, read, update and delete. css is used to set style and design of webpage and javascript is used to add tasks, delete tasks and also to show the status of the task whether it is completed or not....
